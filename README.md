# Quartz Cron 解析器

纯前端 Quartz Cron 表达式解析工具，实时预览未来执行时间。数据不上传服务器，完全在浏览器中运行。

## 使用

访问 GitHub Pages 页面，输入 Cron 表达式，点击"解析"即可查看未来执行时间点。

支持同时对比最多 3 个表达式，可显示相对时间。

## 格式

```
秒 分 时 日 月 周 [年]
```

支持标准 Quartz Cron 语法：`*` `?` `,` `-` `/` 以及英文月份/星期缩写（JAN、MON 等）。

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何构建步骤。
