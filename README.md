# 工具箱

纯前端在线工具集合，运行在浏览器中，无需后端服务。

## 工具

- **Cron 解析器** — 解析 Quartz Cron 表达式，预览未来执行时间。支持同时对比最多 3 个表达式，可显示相对时间。
- **时间戳工具** — 获取当前时间戳（秒/毫秒），时间戳与日期互转，支持一键复制。

## 格式

Cron 表达式格式：
```
秒 分 时 日 月 周 [年]
```

支持标准 Quartz Cron 语法：`*` `?` `,` `-` `/` 以及英文月份/星期缩写（JAN、MON 等）。

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何构建步骤。
