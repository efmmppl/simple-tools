# 工具箱

纯前端在线工具集合，运行在浏览器中，无需后端服务。

## 工具

- **IP 信息** — 查询本机公网 IP 地址及网络位置信息，支持显示内网 IP。
- **金价查询** — 国内国际实时金价，自动刷新。
- **正则测试器** — 测试正则表达式，实时匹配高亮，支持捕获组查看与各种标志位（g/i/m/s/u/y）。
- **Base64 编解码** — 文本与 Base64 互转，支持文件编码。
- **今日热榜** — B站/贴吧实时热搜榜单。
- **文字冒险** — 神秘古堡交互式剧情游戏。
- **数独** — 九宫格逻辑推理，支持多难度自动出题。

## 格式

Cron 表达式格式：
```
秒 分 时 日 月 周 [年]
```

支持标准 Quartz Cron 语法：`*` `?` `,` `-` `/` 以及英文月份/星期缩写（JAN、MON 等）。
